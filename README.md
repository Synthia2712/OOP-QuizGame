# 🎮 Quiz Game Web Application (Java OOP + Spring Boot)

## 📌 Overview

This project is a **Quiz Game Web Application** built using **Java, Object-Oriented Programming (OOP), and Spring Boot**.
It allows users to take a fun, interactive quiz with **multiple difficulty levels**, while demonstrating core OOP concepts in a real-world application.

---

## 🚀 Features

* 🎯 Multiple choice quiz questions
* 👤 Player name input
* 📊 Score tracking
* 🌐 Runs on localhost (web-based interface)
* 🧱 Clean OOP structure with separate classes

---

## 🛠️ Tech Stack

* **Java (JDK 17+)**
* **Spring Boot**
* **Maven**
* **Thymeleaf (for frontend rendering)**
* **HTML/CSS (basic UI)**

---

## 🧩 OOP Concepts Used

* **Encapsulation** → Question and Player classes
* **Abstraction** → Quiz logic handled separately
* **Modularity** → Code divided into multiple classes
* **Object Interaction** → Quiz interacts with Player and Question
